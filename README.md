📓 Journal App (Spring Boot Backend)

A backend REST API for a **Journal Application** built using **Spring Boot**.
This project demonstrates modern backend development practices including authentication, database integration, caching, messaging, and cloud-ready architecture.

🚀 Features

* User authentication and authorization using **JWT**
* Secure APIs using **Spring Security**
* Journal entry creation, update, and deletion
* MongoDB database integration
* Redis caching for improved performance
* Email notification service
* OAuth authentication support
* Kafka messaging integration
* YAML based configuration
* Unit testing with Mockito
* Code quality analysis using SonarQube

🛠 Tech Stack

* Java
* Spring Boot
* Spring Security
* MongoDB
* Redis
* Kafka
* JWT Authentication
* OAuth2
* Mockito (Unit Testing)
* SonarQube
* YAML Configuration
* Maven

2. Navigate to project

3. Run the application
